# Driver Drowsiness Detection & Emergency Alert System

## 🚨 Problem
Driver drowsiness is a major cause of road accidents. Early detection can prevent serious incidents.

## 💡 Solution
This system uses computer vision to monitor driver behavior and detect fatigue in real time.

## ⚙️ Features
- Eye Aspect Ratio (EAR) detection
- Head movement tracking
- Real-time alert system
- Emergency SMS using Twilio

## 🛠 Tech Stack
Python, OpenCV, MediaPipe, NumPy, Arduino, Twilio API

## ▶️ How to Run
1. Install required libraries:
   pip install opencv-python mediapipe numpy
2. Run the script:
   python main.py
